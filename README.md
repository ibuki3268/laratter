## 実装した内容

- Tweet のブックマーク機能を作成した．
- ログイン後のDashbordの画面の作成をした．
- 各画面のUIに手を加えた

## デプロイ先の URL または画面収録したファイル名

エンジニアカタパルトphase1.mp4
## 操作手順

1. 画面上部のナビゲーションの「ブックマーク」をクリックするとブックマーク画面に移動する．ブックマーク画面では Tweet 一覧からブックマークに追加したTweetが表示される．
2. Tweet一覧に移動し、「保存」ボタンを押す．
3. Tweetの表示が「保存」?「保存済み」になりブックマークに遷移すると保存したツイートが表示されている．

## 工夫した点

- 実際に使われているアプリのような直感的に操作できるように各機能の名前の前にアイコンを追加した．
- 何も保存されていない画面でただ空白になるのではなく一覧に遷移するボタンも配置した.

## 苦戦した点

- ブックマークに保存されているときの条件分岐がすぐにはうまくいかなかったのとアイコン形成、表示にあたって広範囲に編集を加えたため苦戦した．
